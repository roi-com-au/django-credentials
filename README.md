# Django Credentials

Storage and retrieval of access credentials.

## Requirements

```bash
pip install -r requirements/base.txt
```

## Installation

```bash
pip install git+git://github.com/roi-com-au/django-credentials
```

## Testing

Use `tox`. `pip install tox` followed by `tox` will do it.
